## Audit Fields

Each concept should be scored before production:

| Field | Score |
| --- | --- |
| Premise clarity | 1-5 |
| Sincerity score | 1-5 |
| Mundane/absurd contrast | 1-5 |
| First-frame strength | 1-5 |
| Final-hold potential | 1-5 |
| Seedance feasibility | 1-5 |
| IP safety | Pass / Revise / Blocked |
| Decision | Greenlight / Revise / Kill |

## Required Output For Greenlit Ideas

Every greenlit idea must produce:

- one-sentence hook
- 3-shot Seedance structure
- first-frame description
- final-hold description
- safety/IP note
