# Absurd Sincere Cinema Content Plan

This folder is the operating plan for an X/Twitter profile focused on Seedance 2.0 videos in one specific lane: **Absurd Sincere Cinema**.

Core profile line:

```text
Impossible scenes played completely straight. Prompt in the replies.
```

## Definition

Absurd Sincere Cinema means strange, high-concept premises performed with total emotional seriousness, premium cinematography, grounded physical detail, and no wink at the camera.

The joke is not sketch comedy. The joke is the contrast between:

- an impossible character or situation
- a painfully mundane institution
- prestige-drama execution
- sincere emotional stakes

## Content Promise

Each flagship post should make a viewer think:

```text
I know exactly what this is in one second, and I need to see how seriously they played it.
```

Prompt/workflow education stays in replies and postmortems. The main feed leads with the premise and the video.

## Core Format

Every flagship concept uses the same structure:

1. **Impossible premise**: one strange archetype or situation.
2. **Mundane institutional setting**: bank, DMV, HR, court, airport security, therapy office.
3. **Emotionally sincere performance**: grief, shame, resignation, dignity, dread, longing.
4. **Cinematic shot-script**: 3 beats across 15 seconds.
5. **Final hold**: the last second lands as an image, not a joke punchline.
6. **Prompt in replies**: reusable Seedance structure after the video.

## Absurd Sincere Gate

Before production, every idea must pass this gate:

- Can the premise be understood in one sentence?
- Does the first frame communicate the absurdity immediately?
- Can it be played emotionally straight without becoming parody?
- Does the setting make the impossible character feel trapped in mundane reality?
- Can the final 1-second hold land as cinematic, not just silly?
- Is it original, with no celebrity, franchise, or IP dependence?

Reject ideas that become sketch comedy, meme parody, generic fantasy, or prompt-engineering content without a strong video premise.

## Production Cadence

Work idea-bank first:

1. Audit all 20 ideas.
2. Choose 8-12 greenlit ideas.
3. Produce 3 flagship videos per week.
4. Post prompt autopsies only for the strongest outputs.
5. Retire ideas that fail the gate or lose the sincere tone during generation.

## Safety Rules

- No real-person likenesses.
- No public-figure names in generation prompts.
- No franchise costumes, characters, settings, or direct visual references.
- No copied choreography.
- No readable logos or third-party marks.
- No captions or in-scene explanatory text as a crutch.

## Files

- `idea-bank.md`: the 20 Absurd Sincere Cinema concepts and first production priority.
- `30-day-calendar.md`: production cadence centered on auditing and generating flagship videos.
- `post-templates.md`: X copy templates for premise-first posts and prompt replies.
- `production-tracker.md`: audit and performance tracker.
