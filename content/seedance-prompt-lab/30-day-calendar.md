# Absurd Sincere Cinema Production Calendar

Start immediately with concept selection. This calendar intentionally skips the previous launch week and does not require daily filler posts.

## Cadence

- 3 flagship Seedance videos per week.
- 1-2 support posts per week only when there is a strong output, prompt autopsy, or failure lesson.
- Weekly review decides which ideas continue.

## Weekly Review Questions

1. Which premise was understood fastest from the first frame?
2. Which video felt most sincere instead of sketchy?
3. Which final hold was strongest?
4. Which prompt produced the least drift?
5. Which idea should become a recurring format?
