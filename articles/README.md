# X Content Calendar

Daily X content now lives inside each experiment folder so code, config, outputs, and launch materials stay together.

Current entries:

- `experiments/001-seedance-facs/content/`

Experiment content folders should include:

- `README.md` for the summary, media paths, and publishing checklist
- `launch-post.md` for the main X post and reply copy
- `x-article.md` for the long-form X Article draft

Generated media stays in `outputs/<experiment-slug>/` and is referenced from the experiment content README instead of committed.
