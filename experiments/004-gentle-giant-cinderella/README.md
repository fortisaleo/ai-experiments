# Experiment 004: Gentle Giant Cinderella

Status: storyboard spec and prompt/config draft.

This experiment develops a sincere gothic fairytale storyboard for the public pitch
"Dave Bautista as Cinderella," but the production subject is a fictional original
gentle giant. The reference image in `workflows/bautista-image.jpeg` is treated as
the character/body reference for scale, strength, posture, and physical presence.
It is not treated as Dave Bautista or any real celebrity.

First pass deliverables:

- `storyboard.md`: story, visual rules, geography, and 12-panel breakdown.
- `config.json`: Nano Banana 2 prompt/config draft for a single 12-panel director board.

No generated media should be produced until the storyboard and prompt are reviewed.
